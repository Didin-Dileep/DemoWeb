# NayePankh Foundation Website

## Project Overview
This project is a modern, responsive, single-page website for NayePankh Foundation, a non-profit organization focused on community upliftment, volunteering, education, health awareness, and social impact.

The site is built with pure HTML5, CSS3, and vanilla JavaScript. It includes a mobile-first layout, dark/light mode, smooth scrolling, scroll animations, form validation, and accessible navigation.

## Features
- Hero section with clear call-to-action
- About, mission, vision, programs, volunteer, gallery, testimonials, and contact sections
- Responsive navigation menu
- Dark/light theme toggle with local storage support
- Smooth scrolling and fade-in-on-scroll effects
- Sticky navigation bar and scroll-to-top button
- Loading animation for a polished first impression
- JavaScript form validation for volunteer registration
- SEO-friendly semantic HTML structure

## Installation Steps
1. Download or clone the repository.
2. Open the project folder in your browser or in VS Code.
3. Launch `index.html` directly in a browser, or use a local preview server.

## Deployment Instructions
1. Upload all files to your hosting provider or static site platform.
2. Make sure `index.html`, `style.css`, and `script.js` are in the root folder.
3. Set `index.html` as the default entry file.
4. Verify the site on mobile, tablet, and desktop breakpoints after deployment.
